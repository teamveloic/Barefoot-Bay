Forum Media Test Tools

This page provides quick access to forum media testing and diagnostic tools for the Barefoot Bay community platform.

Note: These tools are intended for debugging forum media upload and URL formats.

The main test page is in the tmp_debug folder, but you can access it directly using the button below.

Access Forum Media Debug Tool View System Info (JSON)

API Endpoints Reference

  • /api/forum-media-test/system-info - Get system information and storage status
  • /api/forum-media-test/debug/:format/:filename - Test URL formats for forum media
  • /api/forum-media-test/post-media/:postId - Analyze media URLs in a forum post
  • /api/forum-media-test/test-formats/:filename - Test multiple URL formats for a specific file
  • /api/forum-media-test/direct-upload-auth - Get authorization for direct Object Storage uploads
